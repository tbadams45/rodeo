The provided scoring script can be used to generate the score locally. Explore the provided file
ground_truth_2017_2018.csv to find out the expected format of the ground truth file.

Usage: 

python3 scorer.py <target dates> <path to ground truth file> <path to solution file>

Example:

python3 scorer.py 2018-07-01 ground_truth_2017_2018.csv sample_output_2018-07-01.csv
